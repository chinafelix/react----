# react 计算器
基于create-react-app的计算器实现

## #前端页面用react编写

### #RUNNING
```
  cd react-calculator
  
  cnpm install
  
  yarn start
```

## #后端服务用 egg 编写

### #RUNNING

```
    cd egg-calculator
    
    cnpm install
    
    // development模式
    
    npm run dev
    
    或者
    
    // 生产模式
    
    npm start  
```

### #tips
  后端服务可有可无，如果不需要，可以把{app/controller}里的代码复制到前端页面里执行
